# Doc de la mise en place de Weebo 5 sois la futur production

Weebo 5 est le remplaçant de Weebo 4, qui est l'actuelle production.

Weebo 4 étais basé sur une stack K3S, tandis que Weebo 5 est basé sur une stack Talos. Beaucoup de composants seront repris de Weebo 4 mais aussi de WeeboSI, mais d'autres seront remplacés par des alternatives plus modernes et performantes.

L'objectif est de fine tuner au maximum la stack pour qu'elle soit le plus performants possible, tout en étant le plus maintenable possible et sécurisée.

Une grande partie de cette doc est voué a migrer sur mon [blog personnel](https://maxleriche.net), mais je la laisse ici pour le moment pour que les personnes intéressés puissent suivre l'évolution de Weebo 5 et voir les choix techniques qui ont été fait.
